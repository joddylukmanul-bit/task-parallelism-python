# Task Parallelism in Python

Program ini menggunakan ThreadPoolExecutor untuk menjalankan beberapa tugas berbeda secara paralel.

## Tasks:
- Hitung rata-rata
- Cari nilai tertinggi
- Klasifikasi nilai
- Simulasi simpan data

## Cara menjalankan:
python TaskParallelism.py

## Output
Program akan menampilkan hasil dari setiap task yang dijalankan secara paralel.
